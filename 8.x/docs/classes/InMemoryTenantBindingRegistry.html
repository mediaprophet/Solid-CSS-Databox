    Class InMemoryTenantBindingRegistry

    In-memory reference implementation of TenantBindingRegistry. Program-bound facts are process- local configuration; a production deployment swaps in a durable, access-audited store behind the same interface without changing the invariant. Four indexes (by tenant, origin, audience, service identity) each resolve in one hop; every reverse index maps a fact to exactly one tenant.
