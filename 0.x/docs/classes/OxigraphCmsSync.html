    Class OxigraphCmsSync

    Bounded opt-in bridge from canonical Solid CMS RDF writes to an Oxigraph hydration executor.

    This helper deliberately reads after write from the Solid ResourceStore and only replays named-graph replacement updates. It is safe to disable for vanilla Solid mode and safe to rebuild from pod resources later.
