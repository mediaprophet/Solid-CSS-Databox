    Class InMemoryDataboxModuleRegistry

    In-memory reference implementation of DataboxModuleRegistry. Installed manifests and their enabled flags are process-local configuration; a production deployment swaps in a durable, WAC-protected store (module state committed as Solid resources) behind the same interface without changing this contract.
