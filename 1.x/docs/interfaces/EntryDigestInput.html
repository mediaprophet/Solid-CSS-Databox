    Interface EntryDigestInput

    The fields of a LedgerEntry that its entryDigest is computed over (everything except the digest itself). Kept as a distinct type so computeEntryDigest can be called both when appending (no digest yet) and when re-verifying (recomputing an existing entry's digest).

    interface EntryDigestInput {
        outbox?: OutboxRecord;
        prevDigest: string;
        record: AuditEvidenceRecord;
        recordedAt: string;
        sequence: number;
        tenantId: string;
    }
