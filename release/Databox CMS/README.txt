Databox CMS
===========

Package: windows (x64)

1. Double-click "Databox CMS Setup.exe".
2. Setup downloads a private, checksum-verified Node.js runtime and installs the locked application dependencies.
3. Setup starts Databox from the notification area. Use the tray icon to open the admin panel, view logs, or start and stop the server.

The first setup needs an internet connection. It does not use or modify a system-wide Node.js installation.
